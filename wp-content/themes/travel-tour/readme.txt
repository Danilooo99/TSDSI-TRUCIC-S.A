This is one of the best WordPress theme designed for travel blogging, tour operator, travel bloggers, bed and breakfast, hotel reservation, tourism, ticketing, booking, small medium and large scale organisations etc.

== Installation ==
	
1. In your admin panel, go to Appearance > Themes and click the Add New button.
2. Click Upload and Choose File, then select the travel-tour.zip file. Click Install Now.
3. Click Activate to use your new theme right away.
4. Create categories, pages and post
7. Select category needed for the homepage
